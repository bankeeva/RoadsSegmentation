# Roads Segmentation

В данной работе решается задача семантической сегментации спутниковых изображений для выделения дорог на датасете **Massachusetts Roads Dataset**.  
Ссылка на датасет с Kaggle: https://www.kaggle.com/datasets/balraj98/massachusetts-roads-dataset

Была использована архитектура **U-Net** с предобученным энкодером ResNet50.

Одной из ключевых особенностей датасета является дисбаланс классов: пикселей, относящихся к дорогам, меньше, чем пикселей фона. Для частичной компенсации этого дисбаланса использовалась комбинированная функция потерь, объединяющая Focal Loss и Dice Loss:
```python
loss = criterion_focal(pred, label) * 0.2 + criterion_dice(pred, label) * 0.8
```

Дополнительно была выполнена предобработка масок. Поскольку в датасете встречаются изображения с белыми областями, для таких участков значения пикселей в маске приводились к 0, чтобы модель не предсказывала дороги на белом фоне. Затем маски были бинаризованы, а изображения, не содержащие дорог, исключены из обучения. Для тренировочного набора применен кроп по bounding box целевых объектов с последующим приведением всех изображений и масок к единому размеру 1000×1000 пикселей.

**Test scores:**
- **IoU:** 0.6623
- **Dice:** 0.7936
- **Pixel Accuracy:** 0.9793
