# Roads Segmentation

В данной работе решается задача семантической сегментации спутниковых изображений для выделения дорог на датасете **Massachusetts Roads Dataset**.  
Ссылка на датасет с Kaggle: https://www.kaggle.com/datasets/balraj98/massachusetts-roads-dataset

Была использована архитектура **U-Net** с предобученным энкодером ResNet50.

Одной из ключевых особенностей датасета является сильный дисбаланс классов: пикселей, относящихся к дорогам, значительно меньше, чем пикселей фона. Для частичной компенсации этого дисбаланса использовалась комбинированная функция потерь, объединяющая Focal Loss и Dice Loss:
```python
loss = criterion_focal(pred, label) * 0.2 + criterion_dice(pred, label) * 0.8
```

Дополнительно была выполнена предобработка масок. Поскольку в датасете встречаются изображения с белыми областями, для таких участков значения пикселей в маске приводились к 0, чтобы модель не предсказывала дороги на белом фоне.

**Test scores:**
- **IoU:** 0.6016
- **Dice:** 0.7513
- **Pixel Accuracy:** 0.9657
